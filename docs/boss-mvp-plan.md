# W1 Forest Boss「Lil' Slugger」MVP 規劃 v2

依據影片《Super Meat Boy - Boss (World 1/Forest) - Lil' Slugger: Don't Run!》（38 秒）+ 13 張完整截圖（編號 1–13，依時間序）。

## 1. Boss 戰還原（已依截圖修正）

- **Boss 真面目**：Dr. Fetus 駕駛的巨型「鏈鋸步行機甲」——上方駕駛艙、前方水平大鏈鋸、前側大圓鋸輪、後方兩支關節腳（截圖 1、2、12 可見全貌）。v1 誤寫為絞輪怪。
- **流程**：Boss 自左側燃燒山丘出發（截圖 1，腳下有火焰）→ 持續向右步行追擊 → 玩家沿地面路線向右逃，閃避鋸柱、吊鋸、石門 → 終點玩家沿階梯爬上右側高台 → Boss 衝撞高台下方石牆、前傾翻覆起火（截圖 13）→ 過關。
- **修正**：結尾不是 v1 假設的「墜坑」，而是**撞牆翻覆起火**。
- **規則**：碰到 Boss（鏈鋸或機身）即死；死亡後整段重來（玩家回出生點、Boss 重置）。
- **速度**：影片標題證實不衝刺可逃 → Boss 速度 < walk 300 px/s（gameplay.json 實際值），初值 **280 px/s**。

## 2. 關卡尺寸與節奏

- 追擊約 30 秒 × walk 360 px/s → 路線長 ≈ 11,000 px。建議 map **11500 × 1600 px**。
- 主路線為單層地面（截圖中 Meat Boy 幾乎全程貼地）+ 上方高崖當背景與壓頂。
- 終局演出：玩家進入終點區後 Boss 切換加速衝刺，撞牆完成翻覆（重現截圖 12→13 的節奏）。

## 4. 障礙物型別

- **固定鋸**（柱上 / 吊掛 / 斜桿）：渲染物 + 死亡區，沿用現有 deathZones + buzzsaw 動畫幀即可
- **雙鋸旋轉桿**：stretch，MVP 先靜態
- **跳坑**：坑底死亡區
- **Boss**：兩個 AABB——機身 + 前伸鏈鋸（鏈鋸又長又扁，單一大框會誤判），任一重疊即死

## 5. Boss 程式設計（取代 v1 墜坑版）

新增 `src/AppBoss.cpp`。狀態機：

```
WAITING(startDelay) → CHASING(等速 280 向右)
  → RUSHING(玩家 x > rushTriggerX 後加速 700) → CRASHED(抵 crashX：停止、換撞毀燃燒圖)
```

- `BossConfig`（gameplay.json "boss"）：speed、rushSpeed、startDelayMs、visualSize、zIndex、bodyOffset/Size、sawOffset/Size、animFrames、animIntervalMs
- `BossState`：object、phase、waitTimerMs
- `LevelConfig::boss`：`enabled`、`spawn`、`rushTriggerX`、`crashX`（步驟3 起改由 TMX object layer 解析）
- 碰撞：CHASING / RUSHING 判死（機身+鏈鋸雙 AABB）；WAITING / CRASHED 不判死
- 死亡：`RespawnPlayer()` 內呼叫 `ResetBoss()`，任何死因都整段重來
- 攝影機沿用玩家跟隨

## 6. 素材

- **截圖 12（完整行走姿）、13（撞毀燃燒）可直接裁切去背**做 `lilslugger.png` / `lilslugger_crash.png`；行走動畫先用整體 bob + 腿 2 幀；火焰用色塊動畫後補
